Moxtra SDK and API Release Notes
REST API 3.4.0
Sandbox: 11/07/2016 Production: 11/27/2016
  • Added telephonydomain APIs
      • create telephony domain:
      POST /{partner_id}/telephonydomains
      update telephony domain:
      POST /{domain_id}
      delete telephony domain:
      DELETE /{domain_id}
      get telephony domain:
      GET /{domain_id}
      list telephony domains:
      GET /{partner_id}/telephonydomains
      create telephone numbers:
      POST /{domain_id}/numbers
      update telephone number:
      POST /{domain_id}/numbers/{number_id}
      delete telephone number:
      DELETE /{domain_id}/numbers/{number_id}
      upload telephone number prompt:
      POST /{domain_id}/numbers/{number_id}/upload
  • Updated org APIs with telephony_domain support
      • create/update org:
      POST /{partner_id}/orgs
      get org details:
      GET /{partner_id}/orgs/{org_id}
      get org list:
      GET /{partner_id}/orgs
  • Added get meet info return call-in dial numbers
      • GET /meets/{binder_id}
      GET /meets/{session_key}
  • Added /oauth/authorize with clearsession=true to force oauth relogin
  • Fixed get recurrent meet list issue
  • Removed emtpy meeting in get meet list with tag filter
